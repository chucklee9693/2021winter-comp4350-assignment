# COMP4350 2021W Assignment

### Run with `npm start`

### How to Search

- Enter a tag you want to search then click the search button.

- To search with multiple tags, use a semi-colon to separate each tag. <br />
For example, searching with A;B will return questions that have tags of both A and B.

- Passing more than 5 tags will always return 0 result.

- The search will return a merged list of up to 10 newest questions and 10 most voted questions in the past week related to the tag() sorted by creation date descendingly.

- The response time will be shown below the questions retrieved.
